# Privacy Policy for FrameCheck AI

**Last Updated:** February 2025

## Overview

FrameCheck AI is a Chrome extension that analyzes YouTube videos for credibility assessment using AI. This privacy policy explains what data we collect, how it's used, and your rights.

## Data Collection

### Data We Collect

- **Gemini API Key**: Your personal API key, entered by you, stored locally in your browser
- **YouTube Video Data**: When you analyze a video, we extract:
  - Video title
  - Channel name
  - Video description
  - Transcript (if available)
  - Video frames (for visual analysis)
- **Analysis History**: If you enable the history feature, your last 5 analysis results are stored locally
- **User Preferences**: Language selection and history settings

### Data We Do NOT Collect

- Personal identification information
- Browsing history beyond the current YouTube video
- Data from non-YouTube websites
- Analytics or tracking data
- Usage statistics

## How Data Is Used

- **Video metadata and frames** are sent to Google's Gemini API solely for generating the credibility analysis
- **Your API key** is used to authenticate requests to Google's Gemini API
- **All user data is stored locally** in your browser using Chrome's storage API
- **No data is sent to our servers** â€” we do not operate any backend servers

## Third-Party Services

This extension uses **Google Gemini API** to perform AI analysis. When you analyze a video:

- Video metadata, transcript, and frames are sent to Google's servers for processing
- Google's privacy policy applies to this data: https://policies.google.com/privacy

We do not use any other third-party services or analytics.

## Data Storage and Security

- All user data (API key, preferences, cached results, history) is stored locally in your browser
- Data never leaves your device except when sent to Google Gemini API for analysis
- Your API key is stored using Chrome's secure local storage API
- Cached analysis results are stored locally to improve user experience

## Your Rights

You can at any time:

- **Delete your API key** using the "Delete" button in the extension settings
- **Clear analysis history** using the "Clear" button in the history section
- **Disable history saving** by unchecking the "Save history" option
- **Uninstall the extension** to remove all stored data from your browser

## Data Retention

- Local data persists until you manually delete it or uninstall the extension
- We retain no data on any external servers
- Cached results are stored until you navigate to a different video or clear them

## Children's Privacy

This extension is not intended for children under 13 years of age. We do not knowingly collect data from children.

## Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected in the "Last Updated" date at the top of this document.
